# Amateur Radio Lecture and Practice Exam Portal

An interactive educational platform for learning the fundamentals of amateur radio operation, satellite communication, and ham radio licensing preparation.

## Overview

This portal was developed to provide an engaging and modern learning experience for:
- Engineering students
- Amateur radio operators
- Satellite communication enthusiasts
- Participants preparing for amateur radio exams

The platform combines:
- Interactive lecture material
- Practice exams
- Visual learning aids
- SDR and satellite communication concepts
- Mobile-friendly educational content

---

## Features

- Interactive HTML-based learning environment
- Practice examination system
- Question navigation sidebar
- Responsive design for desktop and mobile devices
- Dark-mode modern interface
- Visual illustrations and diagrams
- Satellite communication and SDR-related educational material
- Arabic and English support (where applicable)

---

## Topics Covered

- Amateur Radio Fundamentals
- Radio Wave Propagation
- Antennas and Feedlines
- Basic Electronics
- Operating Procedures
- SDR Basics
- Satellite Communication
- Amateur Radio Regulations
- Practice Licensing Questions

## How to Use

1. Open the portal homepage.
2. Navigate through the lecture sections.
3. Practice using the interactive quizzes and exams.
4. Review explanations and visual aids.
5. Repeat the mock exams to improve readiness.

---

## GitHub Pages Deployment

This project is designed to be hosted using GitHub Pages.

To deploy:

1. Upload the repository to GitHub.
2. Go to:
   `Settings → Pages`
3. Select:
   - Branch: `main`
   - Folder: `/root`
4. Save the settings.
5. GitHub will generate a public URL for the portal.
